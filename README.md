# 더 웰 서비스 (The Well Service) - 프리미엄 공간 관리 솔루션

본 레포지토리는 **더 웰 서비스**의 브랜드 아이덴티티를 담은 프리미엄 웹사이트 프로젝트입니다.
모든 기능과 스타일이 단일 `index.html` 파일에 통합되어 있어, 별도의 서버 설정 없이도 즉시 브라우저에서 실행 및 배포가 가능합니다.

## ✨ 주요 특징

- **단일 파일 아키텍처**: HTML, CSS, JavaScript(React)가 하나의 파일로 통합되었습니다.
- **프리미엄 디자인**: Tailwind CSS와 커스텀 애니메이션을 활용한 고급스러운 UI/UX.
- **반응형 웹**: 데스크탑, 태블릿, 모바일에 최적화된 레이아웃.
- **인터랙티브 갤러리**: 카테고리별 포트폴리오 확인 및 새 갤러리 업로드 시뮬레이션 기능.
- **견적 신청 폼**: 사용자 친화적인 무료 방문 견적 신청 인터페이스.

## 🚀 시작하기

1. 본 레포지토리를 클론하거나 `index.html` 파일을 다운로드합니다.
2. `index.html` 파일을 브라우저로 엽니다.
3. 모든 기능이 즉시 작동합니다.

## 🛠 기술 스택

- **Core**: React 18 (CDN 기반)
- **Styling**: Tailwind CSS (CDN 기반), Vanilla CSS
- **Logic**: Babel (브라우저 내 컴파일)

## 📁 프로젝트 구조

- `index.html`: 전체 서비스 코드 (HTML/JS/CSS)
- `images/`: 웹사이트에 사용되는 로컬 이미지 자산 (사용 시 경로 확인 필요)
